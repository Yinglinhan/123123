# Padding

**padding:** number

padding的设置会被作用到容器的四周。默认是为0。你也可以选择设置单独的一边的padding，使用paddingTop，paddingLeft，paddingRight和paddingBottom。

```jsx
<Page padding={20} />
```



**paddingTop:** number

设置元素的顶部padding，默认是0

```jsx
<Page paddingTop={20}  />
```



**paddingRight:** number

设置元素的右边padding，默认是0

```jsx
<Page paddingRight={20}  />
```



**paddingBottom:** number

设置元素的底顶部padding，默认是0

```jsx
<Page paddingBottom={20}  />
```



**paddingLeft:** number

设置元素的左边padding，默认是0

```jsx
<Page paddingLeft={20}  />
```
