# 快速开始

#### 安装

Framer API被设计为与Framer X这个软件协作，但是它也是一个很棒的独立的JS的库。你可以通过你电脑的命令行工具使用npm安装它。

```bash
// Install Framer via command line
npm install framer
```

#### 

#### 引入

一旦你安装完成，你就可以把这个库引入到你的文件里

```typescript
// Import from Framer
import { Frame, Scroll, useCycle } from "framer"
```
